# 🎙️ Whisper Transcriber

Десктопное приложение для автоматической транскрибации системного аудио в текст с помощью нейросети OpenAI Whisper.

## 🖥️ Возможности

- Захват системного аудио (то что играет на колонках/наушниках) без микрофона
- Распознавание речи через Whisper large-v3 — топовая модель с отличным качеством для русского языка
- GPU-ускорение через NVIDIA CUDA
- Удобный интерфейс на PyQt5
- Сохранение текста в файл `output.txt`
- Копирование текста в буфер обмена одной кнопкой
- Полностью офлайн — данные не покидают устройство

## ⚙️ Требования

- Windows 10 / 11
- Python 3.10+
- NVIDIA GPU с поддержкой CUDA (рекомендуется)

## 📦 Установка

### 1. Клонировать репозиторий

```bash
git clone https://github.com/your-username/whisper-transcriber.git
cd whisper-transcriber
```

### 2. Установить зависимости

```bash
pip install PyQt5 openai-whisper soundcard numpy
```

### 3. Установить PyTorch с поддержкой CUDA (для GPU)

```bash
pip install torch torchvision torchaudio --index-url https://download.pytorch.org/whl/cu121
```

> Без GPU можно пропустить этот шаг — программа запустится на CPU, но медленнее.

## 🚀 Запуск

```bash
python transcriber_gui.py
```

При первом запуске автоматически скачается модель Whisper large-v3 (~3 ГБ).

## 📖 Использование

1. Запустить приложение — дождаться зелёного индикатора «Готово к записи»
2. Нажать **«Начать запись»** — начнётся захват аудио
3. Воспроизвести нужное аудио на устройстве
4. Нажать **«Остановить запись»**
5. Нажать **«Транскрибировать»** — дождаться результата
6. Текст появится в окне и сохранится в `output.txt`

## 🛠️ Технологии

| Технология | Назначение |
|---|---|
| Python 3.12 | Язык программирования |
| OpenAI Whisper large-v3 | Распознавание речи |
| PyQt5 | Графический интерфейс |
| soundcard | Захват системного аудио (WASAPI loopback) |
| PyTorch + CUDA 12.1 | GPU-ускорение |
| NumPy | Обработка аудио |
